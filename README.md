📚 Banco de Dados — Aprendizagem






Repositório criado para documentar meus estudos e práticas em Banco de Dados e SQL.

Aqui são armazenados scripts, exercícios e exemplos utilizados durante o aprendizado de modelagem e manipulação de dados.

Este projeto funciona como um registro da minha evolução no aprendizado de bancos de dados, além de servir como material de consulta futura.

🎯 Objetivo do Projeto

O objetivo deste repositório é:

📚 Consolidar conhecimentos em Banco de Dados

🧠 Praticar consultas SQL

🏗 Entender modelagem de dados

📂 Organizar scripts e exercícios

💻 Criar um histórico de aprendizado para meu portfólio de desenvolvedor

🧠 Conceitos Estudados

Entre os conteúdos presentes no repositório estão:

Criação de tabelas (CREATE TABLE)

Inserção de dados (INSERT)

Consultas (SELECT)

Atualização de dados (UPDATE)

Exclusão de dados (DELETE)

Relacionamentos entre tabelas

Normalização de dados

Estrutura de bancos relacionais

🛠 Tecnologias Utilizadas

Este projeto utiliza principalmente:

SQL

Sistemas de gerenciamento de banco de dados como:

MySQL

PostgreSQL

SQL Server

git clone https://github.com/guilhermebergamini/Banco-de-Dados---Aprendizagem.git
